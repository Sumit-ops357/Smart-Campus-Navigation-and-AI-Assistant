## Overview
A comprehensive smart campus solution featuring interactive maps, event management, real-time navigation, and an AI-powered assistant.

## Technologies
- **Frontend**: React, Vite, Leaflet (Deployed on Vercel)
- **Backend**: Node.js, Express, MongoDB (Deployed on Render)

## Project Structure
- `/frontend`: React application for the user interface.
- `/backend`: API server handling database connections and AI logic.
